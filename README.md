## 任务 1：多项式求值函数

编写函数 `poly_eval(coeffs, x)`，对输入的多项式进行求解，结果精确到小数点后四位。

- 知识点：函数参数、循环与累加、幂运算。
- 数学概念：多项式求值 $f(x) = a_0 + a_1 x + a_2 x^2 + a_3 x^3 + ... + a_n x^n$。

### 任务 2：计算矩阵的行列式

编写函数 `determinant(matrix)`，实现对输入的方阵求解行列式的值，结果精确到小数点后四位。

- 要求：对输入的矩阵求解其行列式的值；
- 数学概念：行列式的展开计算。

提示：
- 当 $n=1$：返回元素本身
- 当 $n=2$： $ad - bc$
- 否则：按第一行展开（递归调用）

### 任务 3：二维点集的凸包

编写函数 `convex_hull(points)`，实现对输入的二维点集求解其凸包。

- 要求：给定点集 $[(x,y), ...]$，返回按逆时针顺序的凸包顶点列表（首尾不重复）。
- 知识点：排序、函数拆分、不可变返回、边界。
- 数学概念：二维点集的凸包，是指由这些点的所有凸组合组成的新的集合。
